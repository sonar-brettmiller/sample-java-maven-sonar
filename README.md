# sample-java-maven-sonar

Minimal Java/Maven project used to validate the SonarCloud GitHub Actions template.

## Setup
1. Add `SONAR_TOKEN` as a GitHub repository secret.
2. Push to `main` and confirm the analysis appears in SonarCloud.

SonarCloud settings:
- sonar.organization: sonar-brettmiller
- sonar.projectKey: sample-java-maven-sonar
